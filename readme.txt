Actively Scans your wordpress website for top 10 OWASP Vulnerabilities like Injection, Broken Authentication , Cross-Site Scripting XSS, Security Misconfiguration etc.

== Description ==

You will get notified as soon as the scanning is completed. You can scan your site once a day and receive detailed report with POC.
<h4>This plugin actively Scans your wordpress website for top 10 OWASP Vulnerabilities.</h4>
* **Injection.** Injection flaws, such as SQL, NoSQL, OS, and LDAP injection
* **Broken Authentication**
* **Cross-Site Scripting XSS**
* **Security Misconfiguration**
* **Checks for WordPress API Endpoints if Accessible**

<h4>FREE Plugin Feature</h4>
* **Penetration Testing Tool:** Scans your wordpress website for top 10 OWASP Vulnerabilities actively.
* **Email Notification:** After a PenTest is completed, We send a report over email.
* **Scan your site Once a day:**  You can perform Pen Test on your site once every 24 hours

== Installation ==


= From your WordPress dashboard =

1. Navigate to `Plugins > Add New` from your WP Admin dashboard.

2. Search for `miniOrange Penetration Testing Tool`.

3. Install `miniOrange Penetration Testing Tool` and Activate the plugin.



= From WordPress.org =

1. Search for `miniOrange Penetration Testing Tool` and download it.

2. Unzip and upload the `miniOrange Penetration Testing Tool` directory to your `/wp-content/plugins/` directory.

3. Activate miniOrange Penetration Testing Tool from the Plugins tab of your admin dashboard.


== Frequently Asked Questions ==

= Once Activated =

1. Select miniOrange Penetration Testing Tool from the left menu and follow the instructions.

2. You can enter your email. You will be notified once the scanning is completed.
